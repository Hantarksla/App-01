# Momentum AI

Silent Accountability & Goal Achievement Platform.

## Vision

Create a personal growth platform combining:

- Goal management
- Habit tracking
- Silent social accountability
- Apple Watch haptic interaction
- AI coaching

## Technology Direction

- Mobile: Flutter
- Apple Watch: SwiftUI / WatchKit
- Backend: Supabase
- Database: PostgreSQL
- Authentication: Supabase Auth
- Push Notification: Firebase Cloud Messaging
- Subscription: RevenueCat
