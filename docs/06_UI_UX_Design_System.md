# UI UX Design System

## Design Goal

The interface should create motivation through clarity, progress visibility and emotional feedback.

## Design Principles

1. Simple

Users should understand today's progress immediately.

2. Positive

Avoid guilt-based design.

3. Rewarding

Every action should create feedback.

## Main Screens

## Home Dashboard

Shows:
- Today's goals
- Completion percentage
- Streak
- Friend activity

## Goal Detail

Shows:
- Milestones
- Progress timeline
- History

## Friend Dashboard

Shows:
- Avatar
- Status ring
- Progress comparison
- Interaction buttons

## Visual Language

Style:
- Clean
- Premium
- Apple ecosystem inspired

Components:
- Cards
- Progress rings
- Heatmaps
- Achievement badges

## Motion

Use animation for:
- Completion
- Level up
- Friend interaction

## Haptic Feedback

Patterns:

Encourage:
Light vibration

Nudge:
Strong short vibration

Achievement:
Celebration pattern
