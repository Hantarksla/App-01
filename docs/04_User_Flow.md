# User Flow

## First Launch

1. Open app
2. Sign in with Apple
3. Create first goal
4. Complete first check-in
5. View progress

## Daily Flow

Morning:
- View today's goals

During day:
- Complete actions

Evening:
- Check-in
- Write short reflection

## Friend Flow

1. Search friend by UID
2. Send request
3. Accept connection
4. View progress
5. Send encouragement
