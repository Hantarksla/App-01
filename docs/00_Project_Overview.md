# Momentum AI - Project Overview

## Product Vision

Momentum AI is a silent accountability platform combining personal goals, habit tracking, social encouragement and Apple Watch interactions.

## Core Concept

People do not need another todo list. They need a system that helps them continue when motivation disappears.

The product loop:

1. Create goals
2. Track daily actions
3. Visualize progress
4. Friends provide silent encouragement
5. Build long-term consistency

## Core Features

- Goal management
- Milestone breakdown
- Daily check-in
- GitHub-style progress heatmap
- Friend accountability
- Silent nudge interaction
- Competition mode
- AI weekly report
- Apple Watch support

## Technology Direction

Mobile:
- Flutter

Watch:
- SwiftUI / WatchKit

Backend:
- Supabase
- PostgreSQL

Services:
- Supabase Auth
- Firebase Cloud Messaging
- RevenueCat
- AI Provider API

## Development Principle

Avoid rebuilding infrastructure. Use mature managed services and focus engineering effort on user experience and behavior design.
