# Product Requirement Document (PRD)

## 1. Product Name

Momentum AI

## 2. Product Goal

Build a platform that improves user consistency through personal tracking and silent social accountability.

## 3. Target Users

### Startup Builders
Need external motivation during long projects.

### Fitness Users
Need consistency for exercise goals.

### Learners
Need daily progress reinforcement.

## 4. Product Principles

- No traditional social feed
- No chat pressure
- Actions over words
- Positive accountability

## 5. MVP Scope

### User System

- Apple Login
- Email login
- Profile
- Unique user ID

### Goal System

Users can create:

- Goals
- Infinite stage tree (stages and sub-stages, unlimited depth)
- Milestone nodes (one-time, optional deadline)
- Recurring action nodes (daily habits)

Node check-in:

- Done (check)
- Not done (cross): recorded as missed, breaks the streak
- Parent stages aggregate child completion automatically

### Check-in System

Record:

- Completion status per goal node (done / missed)
- Duration
- Notes

### Friend System

- Search users
- Add friends
- View progress status

### Silent Interaction

- Encourage
- Nudge
- Completion notification

### Progress Visualization

- Heatmap
- Weekly statistics

## 6. Phase 2

- Apple Watch application
- AI weekly report
- Advanced analytics
- Competition mode
- Goal Globe:
  - Publish goal trees publicly (opt-in)
  - Browse worldwide goals on a 3D globe
  - Adopt (clone) public goal trees, fully editable after adoption
  - Companion notifications and companion check-in dashboard
  - Nudge inactive companions (preset messages, rate-limited)

## 7. Phase 3

- Team challenges
- Enterprise wellness
- AI personal coach

## 8. Success Metrics

Activation:
- First goal created
- First check-in completed

Retention:
- Day 7 retention
- Day 30 retention

Revenue:
- Subscription conversion
